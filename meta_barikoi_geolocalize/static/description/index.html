Meta Barikoi Geolocalize

Partner geolocalization with batch processing and nearby search

Features

  • Auto Geocoding - Automatically geocode partners on create/update
  • Manual Geocoding - Geocode addresses on demand
  • Batch Processing - Geocode all partners without coordinates
  • Nearby Search - Find partners within a distance threshold
  • Status Tracking - Track geocoding status and history
  • Haversine Formula - Accurate distance calculations

Geocoding Status

Status Description
Pending Not yet geocoded
Success Successfully geocoded
Failed Geocoding failed
Manual Manually set coordinates

How to Use

Manual Geocoding

  1. Open a partner form
  2. Enter the address
  3. Click the Geocode Address button
  4. Coordinates will be populated automatically

Batch Geocoding

  1. Go to Contacts → Batch Geocode Partners
  2. All partners without coordinates will be geocoded
  3. A summary of results will be displayed

Nearby Partners

  1. Open a partner with coordinates
  2. Click Find Nearby button
  3. A list of partners within the distance threshold will open

Configuration

Go to Settings → Barikoi Maps to configure:

  • Auto Geocode Partners - Enable automatic geocoding
  • Nearby Distance Threshold - Set maximum distance in km (default: 10)
Requirements: Requires meta_barikoi_base and meta_barikoi_address_autocomplete modules.
