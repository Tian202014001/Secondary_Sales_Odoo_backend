Meta Barikoi Address Autocomplete

Bangladesh-specific address autocomplete for partner forms

Features

  • Real-time Suggestions - Address autocomplete as you type
  • Bangladesh Address Fields - Division, District, Upazila, Union mapping
  • Auto-fill Fields - Automatically populate all address fields on selection
  • Place ID Storage - Store Barikoi place reference for future use
  • Reverse Geocoding - Get address from coordinates on map click

Bangladesh Address Fields

The following fields are automatically populated when selecting an address:

Field Description
Street Full street address
City City/Area name
ZIP Postal code
Division Bangladesh Division
District Bangladesh District
Upazila Bangladesh Upazila
Union Bangladesh Union

How to Use

  1. Open a partner form
  2. Start typing an address in the Street field
  3. Suggestions will appear after 3 characters
  4. Click on a suggestion to auto-fill all address fields
  5. Coordinates are automatically populated
Requirements: Requires meta_barikoi_base module to be installed and configured.
