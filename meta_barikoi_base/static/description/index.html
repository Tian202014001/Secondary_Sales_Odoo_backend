Meta Barikoi Base

Core API connectivity and configuration for Barikoi Maps integration

Features

  • API Key Management - Configure and validate your Barikoi API key
  • Autocomplete Search - Real-time address suggestions for Bangladesh
  • Geocoding - Convert addresses to coordinates
  • Reverse Geocoding - Convert coordinates to addresses
  • Route Planning - Get directions between locations
  • Nearby Places - Find places around a location
  • Place Details - Get detailed information about a place

Configuration

  1. Go to Settings → Barikoi Maps
  2. Enter your Barikoi API key
  3. Enable the integration
  4. Configure additional options as needed
Note: This module is required by all other Barikoi modules.

API Methods Available

Method Description
autocomplete(query) Search for address suggestions
geocode(address) Convert address to coordinates
reverse_geocode(lat, lon) Convert coordinates to address
route(coordinates) Get route between points
nearby(lat, lon, radius) Find nearby places
